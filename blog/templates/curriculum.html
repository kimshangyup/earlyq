데이터 실무자들이 추천하는 데이터 사이언스 온라인 무료 강의 13개

몇 년 전부터 ‘빅 데이터', ‘머신 러닝' 등의 키워드가 주목을 받으면서 데이터 사이언스에 대한 관심이 매우 높아지고 있다. 관련분야 전공자뿐만 아니라 비전공자들도 data science 분야에 대해 알고 싶어한다. 이렇듯 관심이 높아지고 있는데, 그렇다면 실제 데이터 사이언스 분야의 실무자들은 데이터 사이언티스트의 필요 역량을 무엇이라 생각할까? 과연 어떤 공부를 해야 데이터 사이언스를 이해할 수 있는 것일까?

강의는 많은데, 뭘 들어야 하나?

데이터 사이언스에 관해 공부하고자 한다면, 한 번 쯤 미국의 유명 대학 강의들을 중심으로 빠르게 성장하고 있는 MOOC(Massive Open Online Course) 플랫폼이나 DataCamp, Udemy처럼 부분 무료 강의를 제공하는 플랫폼을 들어 봤을 것이다. 혹은 유튜브와 구글에서 검색만 잘 해도 다양한 무료 강의를 찾을 수 있다.

문제는, 강의가 많아도 너무 많아 좋은 강의를 선별하기 어렵다는 점이다. 같은 내용을 다룬 강의가 온라인에 몇 개씩 있는데, 입문자의 입장에서는 이 중에 어떤 강의를 선택해서 들어야 가장 효과가 좋을지, 혹은 이 강의 다음엔 무엇을 들어야 할지 좋을 지 등 여러 고민이 생긴다.

그래서 실무자에게 직접 물어봤습니다

그래서 데이터 사이언티스트, 데이터 애널리스트, 데이터 엔지니어 등의 직무를 달고 현업에서 일하는 몇 몇 지인에게 직접 물어봤다. “직무와 관련하여 온라인 강의를 들어봤면 어떤 강의를 추천하시나요?” 심층 인터뷰 끝에 실무자들에게 추천받은 강의와 추천하는 이유등을 모아보니 하나의 커리큘럼처럼 구성할 수 있었다.

1. 프로그래밍 언어

데이터 사이언티스트를 꿈꾼다면 프로그래밍 언어를 모를 수 없다. 파이썬, R, 포트란, SAS, SPSS 등 다양한 언어들이 있지만 모두 한번에 공부할 수는 없고, 최근에 많이 사용하는 파이썬과 R 정도로 시작해도 충분하다. 프로그래밍 언어는 강의를 들어서 배우기보다 혼자 많은 실습을 통해 익숙해지는게 중요하다. 아래 강좌들을 가볍게 살펴보고 실습을 많이 해보자.

파이썬
Codeacademy

이전에 프로그래밍 언어를 접한적인 없다면, 코드카데미를 추천한다. 한줄 한줄 적어가면서 쉽게 따라갈 수 있게 구성되어 있다.

Python Numpy는 파이썬에서 빅데이터를 다루기 위한 라이브러리이다. 기본적으로 성능이 빠르지 않은 파이썬을 빅데이터 분석이 가능한 정도로 끌어올린 장본인이다. Numpy를 얼마나 잘 사용하는지에 따라 성능이 천차만별로 달라질 수 있다.

R

2. 통계학

데이터 사이언스가 다양한 분야의 융복합 학문이지만, 그 중 가장 중요한 분야를 꼽자면 단연 통계학이다. 통계학은 오래전부터 데이터를 다루는 학문이고 좁은 의미에서 데이터 사이언스는 큰 데이터를 다루는 통계학에 불과하다. 그러니 기초적인 통계학 지식은 필수이고 더 깊게 공부하는 것을 강력하게 추천한다.

통계학 기초 Udacity ‘Intro to Statistics'
Mit Open Courseware Introduction to Probability and Statistics <이미지 : 위에꺼, 텍스트 : MIT가 제공하는 확률과 통계 입문과정. 신뢰가 가는 웹사이트 디자인이 인상적이다>
통계학 심화 Udacity ‘Intro to Inferential Statistics’ Texas University Linear Algebra - Foundations to Frontiers

3. 데이터베이스

통계학이 데이터 분석에 대한 이론적인 지식을 제공해주는 것이라면, 데이터베이스는 데이터 관리와 처리를 담당한다. 작은 데이터를 다룰 때는 그냥 파일에 데이터를 저장해도 문제가 없지만, 10만, 100만 이상의 데이터를 다루다보면 데이터베이스는 필수이다. ‘Using Database with Python’은 파이썬에 익숙한 사람들이 데이터베이스를 배우기 적절한 강좌이다. 두번째 강좌는 MySQL을 사용해서 빅데이터를 분석하는 강좌로 역시나 데이터베이스 입문에 좋은 강좌이다.

Coursera 'Using Databases with Python' Coursera 'Managing Big Data with MySQL’

4. 데이터 분석 및 시각화

데이터 분석은 데이터사이언스의 코어라고 할 수 있다. 또한 이전의 강좌들을 통해 익힌 지식이 총동원되는 분야이기도 하다. 통계학과 선형대수학은 데이터 분석의 이론적 틀을 제공해주며, 프로그래밍 언어와 데이터베이스를 통해 실제 컴퓨터를 사용한 분석을 가능하게 한다. 때문에, 위의 기초가 부실한 상황에서의 데이터 분석을 공부하는 것은 수박 겉핥기에 불과하다. 데이터 시각화는 분석된 결과를 적절하게 보여주는 것이다. 시각화를 단순히 예쁘게 보여주는 것을 넘어 데이터 속에서 새로운 인사이트를 뽑아낼 수 있도록 해준다.

Udacity ‘Intro to Data Analysis’ <이미지 : Udacity Intro to Data Analysis 캡쳐, 텍스트 : Udacity에 올라온 데이터 분석 기초 과정. Facebook과 mongoDB에서 제공하는 강의> Coursera ‘Data Visualization' Coursera 'Data Processing Using Python'

5. Machine Learning

인공지능이 최근 새로운 화두로 떠오르면서 머신러닝은 가장 핫한 분야가 되었다. 머신러닝은 공학에서 우선적으로 발전시킨 분야로써, 데이터 사이언스를 ‘통계학’과 분리시키는데 큰 역할을 했다. 기존의 통계가 통계학의 이론적 틀을 바탕으로 원하는 계산을 했다면, 머신러닝은 수많은 데이터를 컴퓨터 학습할 수 있게 하는 형태이다. 데이터 사이언스 분야에서 머신러닝의 비중은 아직 크지 않다. 그러나 앞으로 더 커질 것으로 전망된다.
